callback tuxxedo_handler(string, callback)

Handler register This function is a wrapper for registering handlers to various functions, calling this function for registering handlers should be registered using this function or some features may stop working unexpectedly

Parameters

string
The handler to register, can be one of 'error', 'exception', 'shutdown' or 'autoload'

callback
The callback to register to the handler

Returns

Returns a callback, if only the first parameter is set, may also return false on error in any case
