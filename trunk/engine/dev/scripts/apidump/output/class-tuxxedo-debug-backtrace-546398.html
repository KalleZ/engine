Debug backtrace class, this is a redesign of the old and infamous tuxxedo_debug_backtrace() function. The class itself defines as an iterator.
This implementation implements some helper methods on each trace instance to ease debugging even further.
Example: use Tuxxedo\Debug;
foreach(new Debug\Backtrace as $trace) if(!$trace->isException()) continue;
