Parses a uri and dispatches them all into controller, action and parameters. The general pattern for the URI routing in this case is fairly common Assumed default names are Index and index for controller and action, though these can be changed using the methods. Default routes: Follows #4 and #5 based on whether number of pieces is even or odd ( odd = #4, even = #5)
