\Tuxxedo\Registry::register()

Meta information
Availability: Engine 1.0.0
Type: Method
Declared in: library/Tuxxedo/Registry.php
Namespace: \Tuxxedo
Object: \Tuxxedo\Registry
Modifiers: public
Prototype
object register(string, string)

Registers a new instance and makes it accessable through the name defined by the first parameter in the global scope like the example below: Note that the class must be a full path, and not an alias as the class is instanciated inside the registry, not the scope where custom aliasing rules may be defined.

Parameters

string
The name of this instance

string
The class to register, this supports the 'Invokable' interface

Returns

Returns a reference to the created instance

Throws

Exception Condition
\Tuxxedo\Exception\Basic Throws a basic exception in case of failure

Examples

Example #1
<?php

use Tuxxedo\Registry;

$registry = Registry::init();
$registry->register('test', '\Full\Class\Path');

$test->Methodname(); // or $registry->test->Methodname();

?>
