Database utilities helper
This helper assumes the 'db' key is registered to an instance of using the 'setInstance' method.
Parameters used by this class are sent directly as a part of the generated SQL query. Only parameters that specifically says so are escaped using the database escaping mechanism assigned to the database registry instance.
Note that some drivers may not be fully supported, in paticular SQLite is not supported by some of the table operational methods.
